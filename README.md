### Tech Stack

CryptoChimp uses the [T3 Stack](https://create.t3.gg), which includes the following technologies:

- Next.js
- TypeScript
- tRPC
- Prisma
- NextAuth
- PostgreSQL
- Tailwind CSS

### Get started

```bash
git clone https://github.com/MaximilianHagelstam/cryptochimp
cd cryptochimp

npm install
npx prisma db push
npm run dev
```
